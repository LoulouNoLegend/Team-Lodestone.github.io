Why Choose Lodestone


Compatible

Lodestone works both as an app and as a web-app. This means that you can access it from anywhere and on any device.


Fun To Use

Lodestone offers extensive customization options while remaining user-friendly. This includes UI coloring customization through themes and customizable modules. 


Fast & Powerful

 With speed optimization and multi-threading, Lodestone ensures highly efficient performance, minimizing resource usage and eliminating memory leaks.


Add-Ons

Extensions, plug-ins, and extension scripts are all supported within Lodestone. This unlocks unlimited possibilities.


Built-In Resigners

Built-in resigners enable you to access your legacy PS3/Vita worlds without the need for any third-party software. Region conversion is also included!


Accessibility

We've implemented several settings and optimizations for users with visual disabilities. Ensuring accessibility is crucial to creating a world for everyone.


Smart Updates

No more worries about staying up-to-date! The toolset will continually expand and evolve in sync with new Minecraft updates and all users will receive the latest updates simultaneously.


Translations

We take pride in supporting all major languages. Being open-source, we also benefit from contributions from the community.


Debug

Debug like a pro with the built-in debugging features. More updates to debugging tools will come to Lodestone as it grows.

 + 
No Sketchy Servers

We do not hide secret servers or loggers of any kind within Lodestone. All worlds stay private and are not sent anywhere. We are 100% open-source and we highly encourage the community to learn how the toolset works, pull requests are also encouraged!
